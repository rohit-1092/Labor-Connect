# Getting start

Just run the following commands in your terminal to start coding

```bash
$ git clone https://github.com/shahbazhassan42000/mazdoor.git
$ cd mazdoor
$ npm install
```

Then run:

```bash
$ npm run dev
```

The last command automatically runs the API exposed on http://localhost:5000
